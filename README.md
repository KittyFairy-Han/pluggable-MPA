1. npm run build
2. 查看dist文件夹，打包结果应该是如下这样的：
``` code
  
    dist
      └── demo
      |   ├── demo-home.html
      |   ├── demo-about.html
      |   └── assets
      |       ├── js
      |       ├── css
      |       └── img
      └── other
          ├── other-home.html
          └── assets
              ├── js
              ├── css
              └── img


```
3. 本地打开 home.html、about.html。能够正常启动。
